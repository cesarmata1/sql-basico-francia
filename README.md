# SQL-Basico-Francia

Este repositorio contiene ejercicios y prácticas relacionadas con **SQL básico**, realizados como parte de la asignatura o trabajo práctico. El enfoque principal es aplicar consultas SQL para resolver ejercicios sobre una base de datos con un contexto relacionado con Francia.

## Descripción

El proyecto incluye scripts y consultas SQL diseñados para practicar operaciones básicas de bases de datos. Los ejercicios se centran en seleccionar, insertar, actualizar y borrar datos, así como en el uso de cláusulas comunes en SQL.

## Contenido del repositorio

- Archivos `.sql` con consultas y definiciones de tablas.
- Ejercicios prácticos resueltos para reforzar conceptos de SQL.

## Tecnologías utilizadas

- SQL (estructura de consultas y manipulación de datos)
